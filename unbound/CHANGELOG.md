# Changelog

## 1.0.0

- Initial release
- Recursive DNS resolution from root servers
- DNSSEC validation enabled by default
- Forwarding mode with optional DNS-over-TLS
- Configurable access control, cache sizes, and threads
- Pi-hole integration support (port 5335)
- Root hints auto-update from internic
- Multi-architecture support (amd64, aarch64, armv7, armhf, i386)
